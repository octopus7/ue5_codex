# Wall Pose Pack Notes

This pack adds 10 wall-interaction body presets to the V2 demo.

These presets are stored in:
- `Content/Poses/Bodies/`

Preset names:
- wall_relaxed_lean
- wall_back_rest
- wall_tired_lean
- wall_push_stretch
- wall_look_around_corner
- wall_sit_rest
- wall_shoulder_lean
- wall_chill_pose
- wall_launch
- wall_thinking_lean

Preview image:
- `Content/Previews/wall_pose_preview_10.png`

Because the editor widget scans the Bodies directory for `*.json`, these new presets will automatically appear in the existing Body Preset combo box.

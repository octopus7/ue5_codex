# Baseball Bat Swing

- Request: add a baseball bat swing motion mode.
- Display names: `Baseball Bat Swing` / `야구 배트 스윙`
- Type: key-pose based non-looping action.
- Data file: `web/manny_baseball_bat_swing_keyposes.js`
- Viewer integration: `web/manny_viewer.js`, `web/index.html`

## Intent

A heavy right-handed swing: compact coiled stance, front heel plant, hips firing first, fast contact through the strike zone, long extension, and a wrapped follow-through.

## Requirements

- Use a visible baseball bat prop.
- Keep both feet planted, with the rear foot pivoting through the swing.
- Hands should stay close on the handle.
- Motion should feel like the body drives the bat, not only the arms.

## 기록 원칙

- 각 항목은 가능한 한 절대 시각으로 기록한다.
- 최소 단위는 `YYYY-MM-DD HH:MM` 형식을 사용한다.
- 단계 시작, 단계 종료, 승인, 보류, 재작업 지시, generator/evaluator 핸드오프를 기록한다.
- 누가 결정했는지, 무엇을 기준으로 판단했는지를 짧게 남긴다.

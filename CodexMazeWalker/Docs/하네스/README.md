# 하네스 작업 문서

이 디렉터리는 실제 수행용 작업본이다. 기준은 `Docs/하네스seed`에 있고, 이 경로는 작업 진행에 따라 계속 갱신한다.

## 사용 순서

1. `00_브리프.md`로 불변 요구사항을 다시 확인한다.
2. `01_실행계획.md`에서 현재 배치의 범위와 병렬 분리를 정한다.
3. 구현과 판단은 `02_진행로그.md`, `03_검증및리스크.md`에 누적한다.
4. 단계별 결정 시점과 수행 시작/종료 시점은 `06_시간기록.md`에 남긴다.
5. generator/evaluator 분리 운영이 필요하면 `04_에이전트운영.md` 규약을 따른다.
6. 문맥이 깨지면 `05_복원절차.md`에 따라 seed에서 다시 복원한다.

## 복원 명령 예시

```powershell
New-Item -ItemType Directory -Force Docs\하네스 | Out-Null
Copy-Item Docs\하네스seed\*.md Docs\하네스 -Force
```

# 하네스 Seed 기준본

이 디렉터리는 장기 수행 작업을 위한 불변 기준본이다. 실행 중에는 `Docs/하네스seed`를 수정하지 않고, 실제 작업은 `Docs/하네스`에서 수행한다.

## 경로 계약

- `Docs/하네스seed`: 복원용 기준본. 작업 중 불변으로 유지한다.
- `Docs/하네스`: 실제 수행 경로. 진행 중 계속 갱신한다.

## 복원 원칙

- 작업 문맥이 깨지거나, 새 세션에서 처음부터 다시 시작해야 하면 `Docs/하네스seed`의 내용을 `Docs/하네스`로 덮어써서 재시작한다.
- Seed 자체를 바꿔야 하는 경우에는 일반 작업 중 수정하지 말고, 별도 기준 개정 작업으로 다룬다.

## 복원 명령 예시

```powershell
New-Item -ItemType Directory -Force Docs\하네스 | Out-Null
Copy-Item Docs\하네스seed\*.md Docs\하네스 -Force
```

## 파일 역할

- `00_브리프.md`: 반드시 유지해야 하는 목표, 제약, 기술 요구사항
- `01_실행계획.md`: 단계별 실행 계획과 병렬 작업 분리 기준
- `02_진행로그.md`: 진행 사항과 의사결정 로그
- `03_검증및리스크.md`: evaluator 관점 검증 항목과 리스크 기록
- `04_에이전트운영.md`: generator/evaluator/main agent 운영 규칙
- `05_복원절차.md`: 하네스 복원 체크리스트
- `06_시간기록.md`: 단계별 결정 시점, 수행 시작/종료 시점, 핸드오프 시점 기록
